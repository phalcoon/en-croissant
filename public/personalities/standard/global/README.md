# Global Audio Clips

Place global audio clips in this directory for special game situations.

## Files Referenced in config.json

The following files are referenced in the parent `config.json`:

- `brilliant.mp3` - Played when a brilliant move is made
- `blunder.mp3` - Played when a blunder occurs
- `checkmate.mp3` - Played when checkmate happens
- `stalemate.mp3` - Played when the game ends in stalemate

## Format

- **Format**: MP3
- **Sample Rate**: 44.1 kHz recommended
- **Bit Rate**: 128 kbps or higher
- **Duration**: 1-4 seconds

## Optional

These files are completely optional. The personality system will gracefully handle missing files.
